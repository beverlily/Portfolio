Projects


  • MakeupStory

    C# / ASP.NET / MSSQL / HTML5 / CSS3
    Visit the Website View on Github

    Description

    MakeupStory is a makeup CMS that allows users to keep track of the makeup products and makeup brands that they have purchased from. Users can give ratings, recommend products, as well as keep track of when they opened a makeup product. A useful feature of this application is that it automatically calculates the expiry date of makeup products so users know when their makeup products are about to expire or have expired.


    Technology

    This was an individual project built in ASP.NET MVC 5. Helpers were implemented to help reduce redundant code in the views. As well, a search feature was implemented so that users can easily look up their makeup products and brands.
